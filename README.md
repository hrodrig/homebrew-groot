# homebrew-groot

Homebrew tap for [groot](https://github.com/hrodrig/groot) — a Go CLI that collects Kubernetes diagnostics (logs, events, resources, node details) into a timestamped archive.

## Install

```bash
brew install hrodrig/groot/groot
```

## Links

- **Project:** [github.com/hrodrig/groot](https://github.com/hrodrig/groot)
- **Releases:** [groot releases](https://github.com/hrodrig/groot/releases)

The cask is updated automatically on each groot release (GoReleaser).
